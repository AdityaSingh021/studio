# **App Name**: Aditya's Portfolio

## Core Features:

- Experience Showcase: Display work experience with RedDoorz and other relevant positions, highlighting key contributions and technologies used like Swift, SwiftUI, UIKit, and REST APIs.
- Project Portfolio: Showcase personal projects like the Hotel Booking App, RedSeller Partner Platform, and Grow Travel and Rewards App, detailing the technologies and improvements made in each.
- Skills Visualization: Visually represent skills in languages, technologies, and tools (e.g., Swift, Java, SwiftUI, Git, Firebase) with proficiency levels to quickly demonstrate expertise.
- Education Summary: Summarize educational background, including degree, university (NIET Greater Noida), and relevant coursework such as Data Structures, Algorithms, and Machine Learning.
- Achievements Section: List certifications and achievements, such as DSA problem solving on LeetCode and global rankings on CodeChef, to highlight accomplishments.
- AI-Powered Resume Enhancement Tool: Use generative AI as a tool that provides suggestions for improving the clarity and impact of the portfolio content.
- Contact Information: Make it easy for recruiters to contact Aditya.

## Style Guidelines:

- Primary color: Deep Teal (#008080) for sophistication and innovation.
- Background color: Off-White (#F8F8FF) for a clean and modern base.
- Accent color: Electric Purple (#793EF2) to highlight key interactive elements and add a touch of modern style.
- Headline font: 'Montserrat', bold sans-serif, for strong section titles; body font: 'Roboto', regular sans-serif, for readable and clean detailed descriptions.
- Use a modern, single-page layout with clear sections and smooth scrolling for easy navigation.
- Employ sleek, line-style icons to represent skills and technologies, ensuring a professional and stylish look.
- Incorporate subtle animations for section transitions and interactive elements, adding visual interest without being distracting.
